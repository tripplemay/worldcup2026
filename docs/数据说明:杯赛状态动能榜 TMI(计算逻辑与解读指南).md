# 数据说明:杯赛状态动能榜 TMI(计算逻辑与解读指南)

> 面向人类用户的说明文档。对应实现:`src/lib/tmi/`(引擎/类型/常量)、`/api/worldcup/tmi`(接口)、App 底部「状态动能榜」页与比赛详情页的「状态动能」对比块。
> 本文描述的公式、权重、阈值与代码一一对应(常量见 `src/lib/tmi/constants.ts`),代码变更时请同步本文。

---

## 一、这是什么(以及不是什么)

**TMI(Tournament Momentum Index,杯赛状态动能指数)** 回答一个问题:

> **「这支队在本届杯赛期间,状态是在往上走还是往下走?」**

它把三类已有数据合成一个 **[-1, +1] 的动能分**,给所有已登场球队排名:

- **士气**——杯赛期间的战绩涨跌(赢了强队涨得多、爆冷输球掉得多);
- **战术**——杯赛期间每场创造与丢失机会的净差(不看比分看机会质量),**按对手强度校正**;
- **体能**——核心球员近期负荷(**按年龄加权**)+ **旅途/跨时区**带来的惩罚。

**它不是什么(同样重要):**

- ❌ **不是实力排名**。实力由 Elo / 评分体系表达;TMI 只量「开赛以来的变化量」。一支夺冠热门若小组赛踢得平庸,TMI 完全可以排在黑马之后——这正是设计意图。
- ❌ **不进入胜率预测**。App 的胜平负概率由预测引擎(泊松+Elo 融合)独立计算,TMI 是**旁观的观测指标**,两者互不影响。页面上「固定权重观测,不进胜率」的标注就是这个意思。
- ❌ **不是投注信号**。它没有对照过市场赔率,不构成任何下注依据。

---

## 二、在哪里看

1. **「状态动能榜」页**(底部导航):全部已登场球队按总分降序的卡片列表。
2. **比赛详情页 → 「状态动能」块**:对阵双方的 TMI 并排对比。
3. **接口**:`GET /api/worldcup/tmi` 返回完整快照(含各队裸数据,方便核对)。

数据是**纯本地合成**:引擎直接读已有的赛果/射门/评分 JSON 现算,不新增任何外部 API 调用;接口带 5 分钟短缓存,底层数据随每日数据摄取与赛后重算自动更新。

## 三、三个因子怎么算

统计范围:**开赛日(2026-06-11,可配)当天及之后**的比赛。只有在此期间登过场的球队才会上榜。

### 因子 1:士气(影子 Elo 净变化)—— 权重 40%

**思想**:用同一把 Elo 尺子量两次,差值就是杯赛期间的净涨跌。

```
影子 Elo 变化 = 自算Elo(全部历史赛果) − 自算Elo(只算到开赛日前的赛果)
```

- 自算 Elo 从 1500 起步、按日期回放全部赛果逐场更新(基础 K=30);
- **净胜球放大**:大胜涨得更多(净胜 2 球 ×1.5,净胜 3 球以上按 (11+净胜)/8 放大);
- **对手强度天然内置**:赢强队(对方 Elo 高)涨分远多于赢弱队——这是 Elo 机制自带的,不需要额外参数;
- 归一化:变化量 ÷ 50 后夹到 [-1, +1]。**「杯赛期间净涨 50 分」≈ 一路爆冷连胜的极限水平,记满分**。

> 为什么叫「影子」:它不是任何权威机构的 Elo,而是我们用同一套代码、同样的起点自算出来的一对平行评分,专门用来做减法。两次计算用同一把尺,系统误差相互抵消,差值只反映杯赛期间发生的事。

### 因子 2:战术(场均 xG 净胜,按对手强度校正)—— 权重 60%

**思想**:比分有运气,机会质量更诚实。一支 0:1 输球但全场压着打的队,战术分可以为正。

```
战术动能 = 杯赛场均(本队 xG − 对手 xG) + 对手强度校正
对手强度校正 = (杯赛对手的赛前基线 Elo 均值 − 全体参赛队均值) ÷ 300
```

- **对手强度校正(v2 新增)**:对弱旅刷出的 xG 优势和对强队打出的同样优势,不再等价。对手比"平均参赛队"每强 300 Elo,补偿约 1 个 xG 当量(Elo 差 300 ≈ 期望胜率 85% ≈ 大约 1 球优势的量级)。对手强度用**开赛前的基线 Elo**,避免"对手后来涨分、我的历史战术分跟着变"的循环;
- **xG 来源**:优先用 API-Football 的真实 expected_goals;该场缺失时回退射门代理(射正×0.3 + 射偏×0.05);
- **小样本回退**:杯赛有 xG 数据的场次 **少于 2 场**时,改用近期全局 EWMA 评分(`xgFor − xgAgainst`,即近况滚动均值),**不做对手校正**(赛季口径的对手已在长期均值中平均化)。此时页面上该项会标注 **「(赛季)」**;
- 归一化:校正后 ÷ 1.5 夹到 [-1, +1]。**「场均净赢 1.5 个 xG」≈ 碾压级表现,记满分**;
- **透明化**:页面小字给出「对手均 Elo」及括号内的校正量,可手工核对。

### 因子 3:体能(负荷 × 年龄 + 旅途)—— 只减不加,合并封顶 −0.6

体能没有「加分」,只有惩罚(0 或负数),由**负荷**与**旅途**两部分相加、合并封顶 −0.6。

**负荷·首选口径:核心 13 人真实分钟,按年龄加权**(有球员分钟数据时)

```
1. 按累计出场分钟选出全队核心 13 人;
2. 统计他们近 8 天的累计【等效】分钟:等效分钟 = 实际分钟 × 年龄系数
   年龄系数:≤21 岁 ×0.95;22~29 岁 ×1;每高 1 岁 +5%(31 岁×1.1、33 岁×1.2),35 岁及以上封顶 ×1.3;
3. 折算成「满场数」(1 满场 = 11 人 × 90 = 990 分钟);
4. 1 个满场以内 → 无惩罚;每多 1 个满场 → −0.4;封顶 −0.6。
```

直觉:同样的连轴转,30+ 的核心阵容比年轻阵容恢复更慢,惩罚更重(v2 新增)。年龄来自
API-Football 名单(与分钟数据同一球员 id 空间,赛期内低频刷新);**缺年龄的球员按 ×1 计,
年龄表未就绪时与旧口径完全一致**(行为中性回退)。页面小字的「核心年龄」即核心 13 人平均年龄。

**负荷·回退口径:休息天数**(无分钟数据时)

```
距上一场 ≥4 天 → 0;休 3 天 → −0.2;休 2 天 → −0.4;休 1 天 → −0.6。
```

**旅途惩罚(v2 新增,美加墨特供)**:2026 的 16 座场馆横跨约 4,500 公里、3 个时区,
各队的飞行负担差异极大——这届比"年龄"更独特的体能变量。

```
旅途惩罚 = −min(0.2, 距离km/1000 × 0.04 + 跨时区数 × 0.04)
```

- 只看**最近两场杯赛**的场馆(比如 温哥华→迈阿密 ≈ 4,500km + 3 时区 → 封顶 −0.2;休斯顿→达拉斯 ≈ 0.01,几乎不计);
- 只在距上一场 **≤7 天**时计(久休后旅途疲劳视为已消化);
- 场馆按比赛城市映射坐标;城市无法识别时诚实不计(不猜)。页面小字给出「旅途 Xkm · N时区」。

## 四、总分怎么合成

```
总分 = clamp[ 0.4 × 士气分 + 0.6 × 战术分 + 体能惩罚 ,  −1 … +1 ]
```

- 士气分、战术分各自已夹在 [-1, +1],体能 ∈ [−0.6, 0];
- 权重是**固定基准**(0.4 / 0.6),不做自动调参——这是一个观测仪表,可比性比"最优"重要;
- 战术权重更高的原因:xG 是逐场的过程质量信号,比赛果驱动的 Elo 噪声更小、对「当下状态」更敏感。

**量纲感受**:总分 +0.5 以上非常罕见(页面会点 🔥 并绿框标出),通常意味着"赢球内容也好、赛程还不累";0 附近 = 表现与开赛前的自己持平;负分 = 状态在往下走。

## 五、界面怎么读(逐元素)

榜单每张卡片从上到下:

| 元素 | 含义 |
|---|---|
| **左侧序号** | 按总分降序的名次 |
| **右上大数字** | 动能总分(+绿 / −红;>+0.5 加 🔥 与绿框 = 「状态火热」) |
| **形态徽章(v2)** | ⚠️ 赢球但内容差(士气高/战术差,动能可能不可持续)· 💎 内容好于战绩(压着打没赢,潜在低估)· 🔋 高负荷(体能扣分 ≥0.4)。把第六节的典型形态直接标在卡片上 |
| **三条贡献条** | 士气(蓝)/ 战术(绿)/ 体能(琥珀色,只会向左)。中线为 0,向右为正、向左为负。条显示的是**加权后的贡献**(士气×0.4、战术×0.6),三条相加(截断前)≈ 总分,可直接看出「分是哪来的」 |
| **小字裸数据** | `影子Elo +12.3` = 杯赛净涨的 Elo 分;`场均xG净胜 +0.42` = 战术因子**未校正**原始值,带「(赛季)」= 杯赛样本不足回退;`对手均Elo 1580(+0.21)` = 对手基线 Elo 均值与括号内的强度校正量;`旅途 2300km · 2时区`;`核心年龄 28.4`;`休息 X 天`;`杯赛 N 场` |

页面顶部另有可折叠的 **「❓ 怎么读这个榜」** 帮助卡(v2),用一句话解释每个因子与徽章含义——第一次看这个榜先展开它。

**三条贡献条的标尺**是共用的(满格 ≈ 0.6 的贡献),所以条的长短可以跨因子直接比。

## 六、解读指南:几种典型形态

1. **士气高 + 战术高**(两条都长)——最扎实的"状态火热":赢了球,过程也压制对手。
2. **士气高 + 战术低/负**——**警示形态**:在赢球但机会创造不占优(点球、门将超神、运气球)。动能可能不可持续,看淘汰赛对阵时值得打个问号。
3. **士气低/负 + 战术高**——**潜在低估形态**:输了或平了,但每场都创造更多机会。若下一轮对手是形态 2,这是经典的"数据看好冷门"结构。
4. **体能条很长(负)**——刚经历加时/点球大战、3 天双赛、老龄核心连轴转,或一次跨洲飞行(小字看「旅途」)。淘汰赛阶段这条的解释力最强,小组赛首轮大家都是 0。
5. **战术项带「(赛季)」标注**——该队杯赛才踢 0~1 场有数据的比赛,战术分其实是"来杯赛之前的近况",别当成杯赛表现读。

**常见误读提醒:**

- **别把榜首当"最强队"**。TMI 量的是变化率:巨头小组赛轻取弱旅、Elo 涨幅有限,排名中游很正常;黑马连爆冷门时会冲到最顶端。
- **样本极小**。杯赛总共 3~7 场,前两轮的榜单波动会很大,越到淘汰赛后期越有参考价值。
- **体能是代理指标**。分钟×年龄+旅途已覆盖主要可量化变量,但仍不含伤病/情绪/气候适应;它提示风险,不判定结果。

## 七、诚实声明与局限

- **观测性指标**:与胜率预测完全解耦,不构成预测,也从未回测过对市场的预测力;
- **固定权重**:0.4/0.6 与各归一化标尺(50 Elo / 1.5 xG)是拍板的直觉基准,不是数据拟合的产物——好处是全届赛事口径恒定、可比;
- **xG 有代理成分**:部分场次的 xG 由射门数据估算(射正×0.3+射偏×0.05),精度低于真实 xG;
- **更新节奏**:底层数据随每日凌晨摄取与每场完赛后的自动重算刷新,接口另有 5 分钟缓存——不是逐分钟实时。
